What it does

🎬

Record & Replay

Enter a URL, start recording. PerfTrace launches a real Chromium instance, records your session with CDP tracing, and streams it via optional VNC.

📊

Rich Metrics

FPS, CPU, GPU, JS heap, DOM nodes, layout/paint, long tasks, Web Vitals (FCP, LCP, CLS, TBT), and React rerender tracking — all in one report.

🖥️

Desktop App

Electron app with bundled Chromium. Works offline, no cloud required. Mac (universal), Windows, and Linux.

How it works

  1. 1 Enter a URL and optionally enable React rerender tracking.
  2. 2 Start recording — Chromium launches and loads your page.
  3. 3 Interact with the page. Watch live metrics or the VNC stream.
  4. 4 Stop recording and get a detailed performance report.
